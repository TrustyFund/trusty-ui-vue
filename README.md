# trusty-vue

To run:
```bash
$ git clone --recursive git@github.com:TrustyFund/trusty-ui-vue.git (or init and update git submodules after simple clone)
$ npm install
$ npm run dev
```

We are using airbnb eslint preset (https://github.com/airbnb/javascript)
